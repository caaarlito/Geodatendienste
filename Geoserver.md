# Geoserver
## Installieren

Herunterladen, dann Zugriff im Browser: 
```
http://localhost:8080/geoserver/web/?0
user: admin; passw: geoserver
```
## Einrichten

Neuen Arbeitsbereich hinzufügen (z.B. „gdd“,“gdd“)
Datenspeicher -> Datenquelle hinzufügen -> PostGIS
